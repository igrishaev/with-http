# with-http

A powerful macro to stub HTTP calls with a local Jetty server. Declarative,
flexible, and extremely useful.

## Installation

Lein:

```clojure
[com.github.igrishaev/with-http "0.1.0"]
```

Deps.edn

```clojure
{com.github.igrishaev/with-http {:mvn/version "0.1.0"}}
```
